# 🌌 Cosmic Forge

빅뱅부터 별의 핵융합까지, 우주의 원소가 만들어지는 과정을 직접 손으로 조합하며 배우는 인터랙티브 웹 시뮬레이션입니다.

## 소개

**Cosmic Forge**는 빅뱅으로 시작해 양성자·중성자·전자 같은 기본 입자를 마우스로 끌어다 겹쳐서 원소를 조합하고, 만들어진 원소를 모아 별을 탄생시키고, 별의 진화와 폭발을 지켜보며 더 무거운 원소가 생성되는 과정을 체험하는 게임입니다. 실제 천체물리학의 원소 생성 원리(빅뱅 핵합성, 항성 핵융합, 삼중 알파 과정, CNO 순환 등)를 바탕으로 미션과 과학 노트를 통해 자연스럽게 학습할 수 있도록 구성되어 있습니다.

## 주요 기능

- **빅뱅 오프닝**: 버튼을 눌러 빅뱅을 일으키며 게임을 시작
- **입자 조합 캔버스**: 양성자·중성자·전자 입자를 드래그 앤 드롭으로 겹쳐 원소 생성
- **별 생성 및 핵융합**: 원소를 충분히 모으면 별이 탄생하고, 시간이 지나 폭발하며 새로운 원소를 만들어냄 (별에 원소를 추가로 투입해 반응을 연장시킬 수도 있음)
- **도감(Guidebook)**: 발견한 원소를 페이지 넘김 효과와 함께 다시 열람하고, 캔버스로 재소환
- **미션 & 업적 시스템**: 단계별 미션을 따라가며 게임을 진행하고, 조건 달성 시 업적 획득
- **과학 노트**: 각 원소·현상에 대한 실제 과학적 배경 설명 제공
- **자동 저장**: 진행 상황이 브라우저 로컬 스토리지에 저장되어 다음 방문 시 이어서 진행 가능

## 기술 스택

- [React](https://react.dev/) 18
- [Vite](https://vitejs.dev/) 5
- CSS Modules
- 브라우저 `localStorage` 기반 저장

## 프로젝트 구조

```
src/
├── components/
│   ├── opening/      # 빅뱅 오프닝 화면
│   ├── canvas/        # 입자·원소·별 조합 캔버스
│   ├── sidebar/       # 원소 도감, 미션/업적, 과학 노트 패널
│   └── guidebook/     # 도감 팝업 및 페이지 넘김 효과
├── context/           # 전역 게임 상태 (GameContext)
├── hooks/             # 조합, 별 생성, 도감, 진행 상황 관리 훅
├── data/              # 원소, 미션, 업적, 과학 노트 데이터
├── utils/             # 조합 판정, 핵융합 로직, 조건 평가, 저장 유틸
└── styles/            # 전역 테마 스타일
```

## 시작하기

### 요구 사항

- Node.js 18 이상
- npm

### 설치

```bash
npm install
```

### 개발 서버 실행

```bash
npm run dev
```

브라우저에서 안내되는 로컬 주소(기본값 `http://localhost:5173`)로 접속하세요.

### 프로덕션 빌드

```bash
npm run build
```

### 빌드 결과 미리보기

```bash
npm run preview
```

## 게임 진행 초기화

게임 진행 상황은 브라우저의 `localStorage`(`cosmic-forge-save` 키)에 저장됩니다. 처음부터 다시 시작하려면 개발자 도구에서 해당 키를 삭제하거나 브라우저 저장 데이터를 초기화하세요.
